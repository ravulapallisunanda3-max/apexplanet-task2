# Task 2 - Contact Form and Dynamic To-Do List

## Project Title

Smart Contact Form and Dynamic To-Do List

## Description

This project was developed as part of the ApexPlanet Web Development Internship Task-2. The project demonstrates intermediate web development skills using HTML, CSS, and JavaScript.

The application includes a responsive contact form with validation and a dynamic to-do list for task management.

## Features

### Contact Form

* Name Field
* Email Field
* Phone Number Field
* Message Field
* Submit Button
* Form Validation

### Form Validation

* Required field validation
* Email format validation
* Phone number validation
* Error messages for invalid inputs
* Success message after submission

### Dynamic To-Do List

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Dynamic task management

### Responsive Design

* Flexbox Layout
* CSS Grid Layout
* Mobile Responsive
* Tablet Responsive
* Desktop Compatible

## Technologies Used

* HTML5
* CSS3
* JavaScript
